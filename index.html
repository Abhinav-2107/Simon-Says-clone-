Simon Says Game

The Simon Game is a memory challenge. The computer will light up one of the colored buttons in a sequence. Your task is to repeat the sequence by clicking the buttons in the same order.

  • Press any key to start the game.
  • Watch carefully as the buttons flash in a sequence.
  • Click the buttons in the exact same order.
  • If you succeed, the sequence gets longer and harder.
  • If you make a mistake, the game ends and you can restart.

Try to remember as many steps as possible — how far can you go?
